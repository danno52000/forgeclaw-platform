# ForgeClaw Theme

Custom branding and theming for OpenClaw Financial Advisor instances.

## Features

- Financial advisor-focused UI/UX
- Professional color scheme and branding
- FA-specific terminology and language
- Compliance-friendly interface elements
- Client-facing presentation mode

## Assets

- `logo/`: ForgeClaw and advisor logos
- `css/`: Custom stylesheets
- `templates/`: Email and report templates
- `icons/`: Financial industry icons
