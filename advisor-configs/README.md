# Advisor Configuration Templates

Pre-configured OpenClaw settings for Financial Advisors.

## Features

- SEC/FINRA compliance templates
- Risk disclosure configurations  
- Client communication settings
- Portfolio analysis defaults
- Market data source configurations

## Template Structure

- `base-config.yml`: Core advisor settings
- `compliance/`: Regulatory compliance templates
- `skills/`: Enabled skills per advisor tier
- `branding/`: Advisor-specific branding options
