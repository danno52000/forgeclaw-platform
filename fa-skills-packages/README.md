# Financial Advisor Skills

Pre-configured OpenClaw skills specifically designed for Financial Advisors.

## Skills Packages

### Core Skills (Included)
- Portfolio Analysis
- Risk Assessment
- Market Research
- Client Reporting

### Premium Skills (Marketplace)
- Advanced Tax Planning
- ESG Screening
- Alternative Investments
- Compliance Monitoring

## Integration with AdvisorIQ

- Neo4j stock graph access
- Risk scoring API
- Thematic portfolio analysis
- Real-time market data

## Skill Structure

Each skill includes:
- `SKILL.md`: Skill documentation and usage
- `skill.py`: Main skill implementation
- `config.yml`: Skill configuration
- `requirements.txt`: Dependencies
