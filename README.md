# ForgeClaw Platform

One-click OpenClaw deployment platform for Financial Advisors with pre-configured skills and compliance frameworks.

## Architecture

- **forgeclaw-portal/**: Next.js advisor signup and management portal
- **fa-skills-packages/**: Financial Advisor specific OpenClaw skills
- **automation-scripts/**: Northflank API integration and deployment automation

## Quick Start

1. Deploy advisor signup portal
2. Configure Northflank API integration
3. Set up FA skills packages
4. Launch advisor instances via portal

## Revenue Model

- Setup Fee: $499 (one-time)
- Hosting: $149/mo per advisor
- Skills Marketplace: $10-50/skill/mo

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Node.js, FastAPI
- **Infrastructure**: Northflank (Kubernetes)
- **Database**: PostgreSQL, Redis
- **Skills**: OpenClaw + AdvisorIQ integration
